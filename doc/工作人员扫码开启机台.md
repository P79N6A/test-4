**简要描述：** 

- 工作人员核销用户课时

**请求URL：** 
- ` http://xx.com/scan/staff/start/equipment `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|token |是  |string |token   |
|content |是  |string |二维码内容(用户课时核销二维码的内容)   |

 **返回成功示例**

``` 
    {
        "code": 0,
        "msg": "获取成功！",
        "data": []
    }
```
 **返回失败示例**
``` 
    {
        "code": 1,
        "msg": "设备状态出错"
    }
```

 **返回参数说明** 

- 无

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
