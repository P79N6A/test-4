
    
**简要描述：** 

- 小程序扫码启动机台

**请求URL：** 
- ` http://xx.com/scan/start/course `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|token |是  |string |token   |
|order_num |是  |string |订单号   |
|content |是  |string |二维码内容   |

 **返回成功示例**

``` 
    {
        "code": 0,
        "msg": "获取成功！",
        "data": {
            "id": 11,
            "name": "测试课程1",
            "course_id": 7,
            "type": 1,
            "sort": 1,
            "created_at": "2018-08-06 10:53:47",
            "updated_at": "2018-08-06 10:56:15"
        }
    }
```
 **返回失败示例**
``` 
    {
        "code": 1,
        "msg": "设备状态出错"
    }
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|name |string   |最新一节课程名称  |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
